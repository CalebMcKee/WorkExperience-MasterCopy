HTML - Hypertext Markup Language

The backbone of the internet

HTML stands for HyperText Markup Language. Developed by scientist Tim Berners-Lee in 1990, HTML is the "hidden" code that helps us communicate with others on the World Wide Web (WWW).

When writing HTML, you add "tags" to the text in order to create the structure. These tags tell the browser how to display the text or graphics in the document. For example, the following document has a simple layout (structure). Notice there are three major parts: a heading, two paragraphs and a bulleted list. 

                    
                    
<html>
    <h1>This is heading text</h1>
    <p>This is a section of paragraph text</p>
    <p>This is another section of paragraph text :)</p>
    <ul>
    <li>first listitem</li>
    <li>second listitem</li>
    <li>third listitem</li>
    </ul>
</html>

The code above will create the following output when viewed in a web browser.

The tools you need

Fortunately, HTML is written in plain text. That means you aren't required to spend lots of money on editing software to create HTML. All you need to create a document is a simple text editor. Every operating system comes with a text editor pre-installed(If you are using Windows, it's Notepad. If you are using an Apple computer, it's TextEdit). One of the most popular free text editors available right now is Sublime Text, which is available for Windows and Apple machines.

The basics

As with everything code-related, there are rules which we as developers must abide by. In HTML, the rules are fairly simple. For starters, HTML tags are always surrounded by what are called angle brackets < and >.

For example: 

                
                
<p>This is how to construct a paragraph in HTML</p>

Elements

The words/letters between these two angle brackets are called elements. These are the coded commands within HTML. Elements tell the browser how to display the web page.
