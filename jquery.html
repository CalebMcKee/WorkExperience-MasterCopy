JQuery

Overview

JQuery is a JavaScript library which provides additional functionality that often makes access elements on the page much easier for the developer as well as providing other functionality which makes coding a lot easier. For our needs we will only be using a very small section of the tools that JQuery makes available to us, but there is plenty more functionality available which you can find in the links below.

To use JQuery we need to download and import the file. This has been done for you within the template document but can be seen below. Note that there are different versions of JQuery, the later versions will provide greater functionality - our version is 2.1.1.

<script type="text/javascript" src="js/jquery-2.1.1.js"></script>

Selectors

The core of what we are going to use JQuery for in our project is to select specific or groups of elements. This is done by accessing the Document Object Model (DOM) which is a tree of all the html tags and searching for the ones which we require. It searches by ID or class which are html attributes that can be put onto html tags. It can also search for different types of tags. This can be seen below. Note that the $ symbol is used to denote any JQuery operation

// In the html...
<div id="firstDiv" class="divs"></div>
<div id="secondDiv" class="divs"></div>

// In the javascript...
$("#firstDiv"); // This will return only the first div
$(".divs"); // This will return both the divs
$("div"); // This will also return both the divs


What to do once you have selected an element

Once you have grabbed the html element(s) that you want, you will want to perform some action on them. This could be as simple as getting the value out of a textbox, or showing or hiding elements on the page:

<script>
   var name = $("#textboxName").val();
   var age = $("#textboxName").val();

   $("#firstDiv").show();
   $("#firstDiv").hide();
</script>

Handling groups of elements

Sometimes when we grab a group of elements using the class selector or by grabbing an element type we want to either find a specific element within that group or perform an action on all the elements. For more information on these check out the JQuery documentation.
