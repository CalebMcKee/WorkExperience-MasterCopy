Javascript

Overview

JavScript (JS) is primarily a client-side development language which runs locally. This means that the code is passed down, along with the html and css, when the page is requested by the server and the browser runs when the code is called. It is used to increase the user experience (UX) of the website and to allow the developer to have greater control over what the website can do. It does this by manipulating the DOM (Document Object Model), which is built from the structure of our HTML. Any element or tag within our document can be found by a unique id or groups of elements by class identifiers.

Sequential operation - JS is a series of commands which run one after the other. It can only do what it is told and in the order given. When writing JS we have to do it either within a separate file which is imported into the html:

<script type="text/javascript" src="filename.js"></script>

or within a code block that is put on the html:

<script>
//Javascript code goes here
</script>

Functions

Code is organised into separate blocks which are shown between curly brackets i.e. { // JS code goes here } We can give blocks of code specific names which allow us to break up the JS into more readable chunks and which can be called from any other point in the code. These blocks are know as functions, and are a core part of JS. For example if we wanted to write a block of code which we would call to print out a custom message to the screen we could do it using a function and a parameter (a variable which is passed into the function so that it can be used.) we would get something close to the following:

<script>
   function customAlert(name) {
      alert(name + " this is a personal message for you");
   }
   customAlert("Barry");
   customAlert("Joan");
</script>


Variables

Variables are a programming concept and are places where data is stored. Once a value is assigned to a variable, it is kept there until the current block of code or function has completed executing.

<script>
   var name = "Barry";
   var integer = 10;
   var person = {name: "James", age: "25", gender:"male"};
   customAlert(name);
   integer = integer + 25;
   customAlert(person.name);
</script>

Note that in that last code block we saw an interesting type of variable which was "person". This is what is known as an object and it is a set of different values stored in a single variable, where each variable is identified by a unique key (name, age or gender in our example). Objects are important in JS and can also be known as JSON.


Condtional (If) statements

Conditional statements, also known as IF statements are ways to allow the code to make decisions at runtime depending on a specific test. If the test is true, then the code within the condtional block is run, however if the test is false then it will jump on to any other test conditions or an else block if one exists. For example, if I had a variable called name and wanted to say that if the name was "Barry" to set an age variable to be 18, but if the name is "Joan", set the age to be 19 and if it is nether to set the age to be 20, we would do it like this:

<script>
   if(name === "Barry") {
      age = 18;
   } else if (name == "Joan") {
      age = 19;
   } else {
      age = 20;
   }
</script>
